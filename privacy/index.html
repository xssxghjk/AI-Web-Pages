Privacy Policy

A hobby site with nothing to hide.

The short version

This site does not collect, store, or transmit any personal data. There are no cookies, no analytics trackers, no user accounts, and no third-party advertising. All data stays on your device.

What is stored and where

Some pages save data in your browser's localStorage so that your work persists between visits. This data never leaves your device.

  • Saved decks — card lists and sideboard notes you build in the Deck Builder. Local only
  • Marathon training log — workout entries and plan data. Local only
  • Trip notes & state — packing and checklist state on trip detail pages. Local only
  • UI preferences — things like selected chart view in Project Stats. Local only
  • Notice dismissed — whether you've dismissed the storage notice banner. Local only

You can clear all of this at any time from your browser's developer tools or site settings.

Cookies

This site sets no cookies whatsoever — first-party or third-party. None

Analytics & tracking

There is no analytics, telemetry, heatmapping, session recording, or any other tracking on this site. None

Third parties

This site loads fonts from Google Fonts (fonts.googleapis.com / fonts.gstatic.com). This means Google's servers receive your IP address as part of serving those font files. Google's privacy policy applies to that request.

The site is hosted via GitHub Pages. GitHub may log request metadata (IP address, browser, timestamp) as part of normal server operation. GitHub's privacy policy applies.

No other external services are used.

Children

This site is not directed at children and does not knowingly collect any information from anyone.

Contact

This is a personal hobby project. If you have questions, open an issue on GitHub.
